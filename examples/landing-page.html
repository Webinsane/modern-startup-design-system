The problem

AI can code. It just can't design.

Every AI assistant produces the same generic UI: plain borders, system fonts, zero personality. You end up with something that works but looks like it was built in 2014. This design system fixes that.

One-prompt setup

Paste the design system into your AI's context. Every component it generates will use the right tokens, spacing, and dark mode support automatically.

Dark mode included

Add data-theme="dark" to the html tag. Every component adapts instantly. No extra CSS needed.

Re-brand in 4 lines

Change four CSS variables and the entire system re-skins: buttons, links, cards, nav, focus rings. The three-layer token architecture does the rest.

48+ components

Nav bars, filter bars, stat cards, schedule views, task boards, tables, forms, modals, alerts, toasts, heroes, pagination, tags and more.

Zero dependencies

Pure CSS with semantic tokens. No build step, no JavaScript framework, no package manager. Works with React, Vue, Svelte, or plain HTML.

Responsive by default

Every component includes mobile breakpoints. Sidebars collapse, grids reflow, nav bars adapt. No extra work for responsive layouts.

How it works

Three steps to a beautiful app

No figma files to translate. No design handoff meetings. Just paste and build.

1

Copy the system

Grab tokens.css and components.css. Paste them into your AI assistant's context window or CLAUDE.md file. That's the entire setup — two files, zero config.

2

Describe what you need

"Build me a team dashboard with stats, a task board, and a schedule sidebar." The AI knows every class name, every token, every breakpoint.

3

Ship it

Production-ready HTML with proper BEM classes, dark mode, responsive layout, and polished visuals. No cleanup pass needed — just deploy.

Example prompt
/* Your AI prompt */ Build a project management dashboard using the Modern Startup design system. Include: - Sidebar nav with dashboard, projects, calendar, and settings icons - Top nav with search bar and user avatar - Stats row showing revenue, active users, sprint progress - Task board with 3 columns: Backlog, In Progress, Done - Schedule sidebar showing today's meetings Use components.css classes. Support dark mode via data-theme="dark".

What's included

Everything your dashboard needs

Colors, typography, spacing, shadows, motion, layout. Three-layer architecture: raw palette → brand slots → semantic tokens. Dark mode overrides only the semantic layer.

48+ components using BEM naming. Sidebar nav, top nav, filter bars, stat cards, schedule cards, task boards, tables, forms, modals, alerts, toasts, heroes, pagination, tags, toggles, avatars, and more.
